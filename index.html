دکمه‌های مدرن

توضیحات فنی دکمه‌ها

دکمه‌های فوق با استفاده از تگ <button> ایجاد شده‌اند. هر دکمه دارای کلاس‌های مختلفی است که ظاهر و رفتار آن را تعیین می‌کند.

کلاس‌های اصلی:

  • .btn: استایل پایه برای تمام دکمه‌ها
  • .btn-primary: دکمه اصلی با رنگ آبی
  • .btn-secondary: دکمه ثانویه با رنگ بنفش
  • .btn-accent: دکمه تاکیدی با رنگ روشن
  • .btn-outline: دکمه با حاشیه و زمینه شفاف
  • .btn-gradient: دکمه با پس‌زمینه گرادیانتی

انیمیشن‌ها:

  • هنگام hover دکمه‌ها به سمت بالا حرکت می‌کنند و سایه ایجاد می‌شود
  • .btn-pulse: ایجاد انیمیشن ضربان هنگام hover
  • .btn-shake: ایجاد انیمیشن لرزش هنگام hover

رعایت اصول سئو:

  • استفاده از تگ معنایی <button> به جای div برای دکمه‌ها
  • اضافه کردن آیکون‌های معنادار با <i> از Font Awesome
  • استفاده از متن توصیفی در دکمه‌ها

منوی ناوبری افقی

  • صفحه اصلی
  • درباره ما
  • خدمات
  • نمونه کارها
  • تماس با ما

منوی آبشاری (Dropdown)

لپ‌تاپ و کامپیوتر گوشی موبایل تبلت لوازم جانبی دوربین

توضیحات فنی منوها

منوها با استفاده از تگ‌های معنایی <nav>، <ul> و <li> ایجاد شده‌اند که برای سئو بسیار مناسب است.

منوی افقی:

  • با استفاده از Flexbox طراحی شده است
  • دارای افکت hover با تغییر رنگ زمینه و حرکت به بالا
  • حاشیه‌های گرد و ظاهر مدرن

منوی آبشاری:

  • با ترکیب position: relative و absolute ایجاد شده
  • منوی آبشاری هنگام hover روی دکمه اصلی نمایش داده می‌شود
  • دارای انیمیشن fadeIn هنگام نمایش
  • آیتم‌های منو هنگام hover دارای پدینگ بیشتر می‌شوند

رعایت اصول سئو در منوها:

  • استفاده از ساختار لیست (ul/li) برای منوها
  • استفاده از تگ nav برای بخش ناوبری اصلی
  • استفاده از لینک‌های معنادار با متن توصیفی
  • اضافه کردن آیکون‌های مرتبط برای هر آیتم منو

نکات طراحی و سئو

بهینه‌سازی سئو

  • استفاده از تگ‌های معنایی HTML5 مانند <header>, <main>, <footer>, <nav>
  • استفاده از meta tags توصیفی شامل description و keywords
  • تعیین زبان صفحه با lang="fa" و جهت متن با dir="rtl"
  • استفاده از تگ‌های heading به ترتیب اهمیت (h1 تا h6)
  • استفاده از متن جایگزین (alt) برای تصاویر (در این صفحه تصویری وجود ندارد)
  • طراحی ریسپانسیو برای نمایش در دستگاه‌های مختلف

اصول طراحی رابط کاربری

  • استفاده از رنگ‌های هماهنگ و کنتراست مناسب
  • افکت‌های hover برای تعامل بهتر کاربر
  • انیمیشن‌های سبک و غیرمزاحم برای تجربه کاربری بهتر
  • فضای خالی مناسب برای خوانایی بهتر
  • حاشیه‌های گرد برای ظاهر مدرن
  • سایه‌های ملایم برای ایجاد عمق
  • استفاده از آیکون‌های Font Awesome برای درک بصری بهتر

تگ‌های مهم استفاده شده

  • <button>: برای ایجاد دکمه‌های قابل کلیک
  • <a>: برای لینک‌ها در منوها
  • <ul> و <li>: برای ساختار لیست‌های منو
  • <div>: برای گروه‌بندی عناصر
  • <i>: برای نمایش آیکون‌ها
  • <code>: برای نمایش کدهای نمونه

تکنیک‌های CSS پیشرفته

  • متغیرهای CSS (Custom Properties) برای مدیریت رنگ‌ها
  • Flexbox و Grid برای چیدمان‌های مدرن
  • گرادیانت برای پس‌زمینه‌ها و دکمه‌ها
  • انیمیشن‌های CSS برای افکت‌های hover
  • مدیا کوئری‌ها برای طراحی ریسپانسیو
  • سایه‌های متن و باکس برای ایجاد عمق
  • ترنزیشن‌ها برای تغییرات نرم
